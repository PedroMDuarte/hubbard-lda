This directory contains a combination of QMC data and Ehsan's data which is 
used to calculate Spi for various values of U, T and n. 

Tha base is all of the QMC data and on top of this I have added a few data
sets from Ehsan.   Below is the list of NLCE added sets

==========

U10:  T=0.72, 0.84, 0.90, 1.00
U14:  T=0.72, 0.84, 0.90, 1.00
